--- layout: post title: First Ottawa Android Developer Meetup categories: - Meetings tags: [] status: publish type: post published: true meta: _edit_last: '2' dsq_thread_id: '' --- Tomorrow is going to be the first monthly meeting for Android developers in the Ottawa-Gatineau area. The meeting is happening at The Code Factory which is at 246 Queen St, right above the Green Papaya. Since this will be the first meeting, we will only be covering what we want the goals of these meetings to be, project ideas, and so on. Feel free to come out with ideas for any projects you would be interested in doing. Some ideas we currently have in mind:
  • Presentations around usage of the Android API
  • Hack-a-thons / Projects
  • Flash talks covering personal projects or new technologies for Android applications
